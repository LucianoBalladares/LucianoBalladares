# Luciano Balladares  
### Analista de Datos en Salud | Tecnólogo Médico (BACIMET)

Profesional con formación en tecnología médica y especialización en informática en salud, enfocado en el análisis de datos administrativos para la evaluación de desempeño del sistema de salud.

Interesado en la intersección entre:
- Gestión de redes asistenciales
- Análisis de datos en salud pública
- Calidad y eficiencia del sistema sanitario

## 🔎 Enfoque actual

Actualmente trabajo en el desarrollo de modelos analíticos orientados a:

- Evaluación de listas de espera NO GES
- Identificación de brechas estructurales en Servicios de Salud
- Análisis de indicadores de desempeño en redes asistenciales

---

## 📊 Proyecto destacado

### Análisis de recuperación de listas de espera NO GES en Chile

- Construcción de dataset longitudinal a partir de fuentes no estructuradas (Glosa 06)
- Definición de indicadores clave:
  - Mediana de espera
  - Velocidad de recuperación
  - Antigüedad extrema
  - Concentración en nivel terciario
- Desarrollo de dashboard interactivo en Power BI
